て形変化

Posted on 2019-09-19 In 日本語

動詞

  • 第I類動詞

    第I類動詞 会(あ)
    買(か)    		 書(か)
    置(お)    		 泳(およ)
    急(いそ)    		 話(はな)
    貸(か)    		 待(ま)
    立(た)    		 死(し) 遊(あそ)
    呼(よ)    		 読(よ)
    飲(の)    		 帰(かえ)
    売(う)    		 形態
    あ段 わ か が さ た な ば ま ら -
    い段 い き ぎ し ち に び み り ます形
    う段 う く ぐ す つ ぬ ぶ む る -
    え段 え け げ せ て ね べ め れ -
    お段 お こ ご そ と の ぼ も ろ -
    音便 っ い い(“) し っ ん(“) ん(“) ん(“) っ て（で）形

    い段 => 音便 + て(で)
    例外: 行きます => 行って

  • 第II類動詞

    第II類動詞 -
    教(おし)えます 教えて
    食(た)べます 食べて
    起(お)きます 起きて
    見(み)ます 見て
    寝(ね)ます 寝て

  • 第III類動詞

    第III類動詞 -
    来(き)ます 来て
    します して

形容詞

  • い形容詞

    ～い　=> ～くて
    おいしい　=> おいしくて

  • な形容詞

    ～な　=> ～で
    元気(げんき)な => 元気で

名詞

～の　=> ～で
学生(がくせい)の => 学生で
雨(あめ)の => 雨で

て形の文型

  • 並(なみ)述(じゅつ)

    京都(きょうと)はお寺(てら)があって、静(しずか)かです。=> (動)
    北海道(ほっかいどう)は寒(さむ)くて、雪(ゆき)も多(おお)いです。=> (い)
    東京(とうきょう)は賑(にぎ)やかで、交通(こうつう)が便利(べんり)です。=> (な)
    彼(かれ)は台湾人(たいわんじん)で、筑波(つくば)大学(だいがく)の学生(がくせい)です。=> (名)

  • 原因、理由

    手紙(てがみ)を貰(もら)って、嬉しいです。=> (動)
    この店(みせ)は美味しくて、人気(にんき)があいます。=> (い)
    ここは交通が便利で、家賃(やちん)が高いてす。=> (な)
    台風(たいふう)で電車(でんしゃ)が止まりました。

    • 方法、手段

    駅まで歩(ある)いて行きます。=> (動)
    駅までバスで行きます。=> (名)
