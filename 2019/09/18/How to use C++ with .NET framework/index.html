How to use C++ with .NET framewok(C++/CLI)

Posted on 2019-09-18

If you want to use C++ on .NET framework , you can follow the steps in this article.

Create a new CLR project


Add > New Item

Select CLR - Windows Form

Setting project properties

Project > Properties

Linker > System > SubSystem

Select WINDOWS(/SUBSYSTEM:WINDOWS)

Linker > Advanced > Entry Point

Enter main

Open .cpp file and paste it in

Unload Project and Reload Project

It is work!
